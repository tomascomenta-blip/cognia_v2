# investigar-tema

Investiga un tema a fondo antes de responder.

## Como proceder
1. Define el tema: usa `anotar` para escribir el tema exacto que se va a investigar.
2. Busca en fuentes principales: usa `http_get` para buscar en sitios web confiables como documentacion oficial, blogs de expertos o foros.
3. Extrae información relevante: usa `leer_archivo` para ver los contenidos descargados y `resumir` para obtener los puntos clave.
4. Busca en otras fuentes: usa `http_get` para buscar en otros sitios web o recursos si es necesario.
5. Compara la información: usa `notas` para anotar las diferencias y similitudes entre las fuentes.
6. Verifica la información: usa `http_get` para buscar confirmaciones o refutaciones de la información encontrada.
7. Forma una conclusion: usa `anotar` para escribir lo que se ha aprendido y las conclusiones finales.

## Reglas
- Usa varias fuentes para contrastar la información.
- No responda de memoria sin investigar.
- Verifica la información antes de formar una conclusion.
- Anota todos los pasos y la información relevante para referirse luego.
