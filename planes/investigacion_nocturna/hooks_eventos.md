# Investigacion: hooks and event driven automation in coding agent CLI tools

Queries ejecutadas (4): `awesome coding agent hooks`, `event driven agent cli`, `automation agent tools`, `coding agent alternatives`

## Resumen

The search results provide several repositories and tools that support hooks and event-driven automation in coding agent CLI tools. Notably, `michielhdoteth/awesome-ai-agent-tools` offers a comprehensive library of AI agent components including hooks and plugins, making it a valuable resource for developers looking to implement event-driven automation. Additionally, `barryroodt/gantry` is a Rust-based standalone LLM agent harness that operates over the CLI and is observed via an NDJSON event stream, which aligns well with event-driven automation principles. The `enfibiotech/rl-intern-` project also implements an agentic reinforcement learning workflow with an event-driven agent loop, providing a practical example of how hooks and event-driven automation can be integrated into coding agents. These tools and models offer developers the necessary components and frameworks to build sophisticated CLI-based automation systems for coding agents.
