## Resumen

The search results provide several relevant benchmarks and evaluation harnesses for autonomous coding agents. The `chrisliu298/awesome-research-agents` repository is particularly noteworthy as it curates a collection of research agents, skill libraries, and benchmarks built around Claude Code, OpenAI Codex CLI, and adjacent coding-agent CLIs for AI/ML research. Additionally, the `arxiv` paper titled "Advances and Frontiers of LLM-based Issue Resolution in Software Engineering: A Comprehensive Survey" discusses the establishment of benchmarks like SWE-bench, which revealed the difficulty of large language models in issue resolution tasks. The `arxiv` paper "SCATE: Learning to Supervise Coding Agents for Cost-Effective Test Generation" focuses on the limitations of autonomous coding agents in automated test generation and introduces a framework to address these issues. Lastly, the `arxiv` paper "Agents: An Open-source Framework for Autonomous Language Agents" provides an overview of language agents and their capabilities, which is essential for understanding the landscape of autonomous coding agents.
